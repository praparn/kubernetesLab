Instruction for Workshop 2.1_ConfigMap_Secret:
Note: This instruction will demonstrate about how to ConfigMap and Secret on Microservice:
====================================================

====================================================
Part1: Container level configuration:
====================================================
1. Get command prompt for access folder workshop 2.1 by command :cd /xxxx/WorkShop_2.1_ConfigMap_Secret
2. Check File on WorkShop

3. Create Namespace by command:
	kubectl create namespace webmicroservice

4. Create ConfigMap by command:
	kubectl create -f webmodule_configmap.yml --validate=false  --namespace webmicroservice	(In case run via github: kubectl create -f https://raw.githubusercontent.com/praparn/kubernetesLab/master/WorkShop_2.1_ConfigMap_Secret/webmodule_configmap.yml --validate=false  --namespace webmicroservice)
	kubectl get configmap --namespace webmicroservice
	kubectl describe configmap/webmodule-configmap --namespace webmicroservice

5. Create Secret by command:
	kubectl create -f databasemodule_secret.yml --validate=false  --namespace webmicroservice	(In case run via github: kubectl create -f https://raw.githubusercontent.com/praparn/kubernetesLab/master/WorkShop_2.1_ConfigMap_Secret/databasemodule_secret.yml --validate=false  --namespace webmicroservice)
	kubectl get secret --namespace webmicroservice
	kubectl describe secret/databasemodule-secret --namespace webmicroservice

6. Create Database Deployment by command:
	kubectl create -f databasemodule_deploy_config.yml --validate=false  --namespace webmicroservice	(In case run via github: kubectl create -f https://raw.githubusercontent.com/praparn/kubernetesLab/master/WorkShop_2.1_ConfigMap_Secret/databasemodule_deploy_config.yml --validate=false  --namespace webmicroservice)
	kubectl get deploy --namespace webmicroservice
	kubectl get rs --namespace webmicroservice
	kubectl get pods --namespace webmicroservice

7. Check Detail of Pods by command:
	kubectl describe pods/<pods name>  --namespace webmicroservice

8. Create WebModbule Deployment by command:
	kubectl create -f webmodule_deploy_config.yml --validate=false  --namespace webmicroservice	(In case run via github: kubectl create -f https://raw.githubusercontent.com/praparn/kubernetesLab/master/WorkShop_2.1_ConfigMap_Secret/webmodule_deploy_config.yml --validate=false  --namespace webmicroservice)
	kubectl get deployment/web --namespace webmicroservice
	kubectl get rs --namespace webmicroservice
	kubectl get pods --namespace webmicroservice

9. Check Detail of Pods by command:
	kubectl describe pods/<pods name> --namespace webmicroservice

10. Setup Environment for Test:
	export Server_IP=<ip of minikube>
	export Server_Port=30500 ==> if need to direct test change to 32500
