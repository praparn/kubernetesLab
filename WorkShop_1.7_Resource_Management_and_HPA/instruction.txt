====================================================

====================================================
Part1: Container level configuration:
====================================================
1. Get command prompt for access folder workshop 1.7 by command :cd /xxxx/WorkShop_1.7_Resource_Management_and_HPA

====================================================
Part 2: Name space level configuration:
====================================================

1. Create Namespace by command:
	kubectl create namespace webtest-namespace

2. Create Quota and attach to Namespace and view describe by command:
	kubectl create -f webtest_quota.yml --validate=false --namespace=webtest-namespace (In case run via github: kubectl create -f https://raw.githubusercontent.com/praparn/kubernetesLab/master/WorkShop_1.7_Resource_Management_and_HPA/webtest_quota.yml --validate=false --namespace=webtest-namespace)
	kubectl describe namespace/webtest-namespace

3. Try to create deployment by command:
	kubectl create -f webtest_deploy.yml --validate=false --namespace=webtest-namespace	(In case run via github: kubectl create -f https://raw.githubusercontent.com/praparn/kubernetesLab/master/WorkShop_1.7_Resource_Management_and_HPA/webtest_deploy.yml  --validate=false --namespace=webtest-namespace)

4. Check completeness of deployment/rs/pods/svc by command:
	kubectl get deployment/webtest --namespace=webtest-namespace
	kubectl get rs --namespace=webtest-namespace
	kubectl get svc/webtest --namespace=webtest-namespace
	kubectl get pods --namespace=webtest-namespace

5. Check full describe of deployment/rs/pod by command:
	kubectl describe deployment/webtest --namespace=webtest-namespace
	kubectl describe rs --namespace=webtest-namespace

6. Create LimitRange attach to Namespace and view descibe by command:
	kubectl create -f webtest_limit.yml --validate=false --namespace=webtest-namespace	(In case run via github: kubectl create -f https://raw.githubusercontent.com/praparn/kubernetesLab/master/WorkShop_1.7_Resource_Management_and_HPA/webtest_limit.yml --validate=false --namespace=webtest-namespace)
	kubectl describe namespace/webtest-namespace

7. Recreate Deployment by command:
	kubectl delete -f webtest_deploy.yml --validate=false --namespace=webtest-namespace	(In case run via github: kubectl delete -f https://raw.githubusercontent.com/praparn/kubernetesLab/master/WorkShop_1.7_Resource_Management_and_HPA/webtest_deploy.yml  --namespace=webtest-namespace)
	kubectl create -f webtest_deploy.yml --validate=false --namespace=webtest-namespace	(In case run via github: kubectl create -f https://raw.githubusercontent.com/praparn/kubernetesLab/master/WorkShop_1.7_Resource_Management_and_HPA/webtest_deploy.yml  --validate=false --namespace=webtest-namespace)

8. Check completeness of deployment/rs/pods/svc by command:
	kubectl get deployment/webtest --namespace=webtest-namespace
	kubectl get rs --namespace=webtest-namespace
	kubectl get svc/webtest --namespace=webtest-namespace
	kubectl get pods --namespace=webtest-namespace

9. Verify configuration on Pods by command:
	kubectl describe pods <pods name> --namespace=webtest-namespace

10. Start burn test cpu and monitor cpu by command:
	kubectl exec -it <pod name> -c webtest md5sum /dev/urandom --namespace=webtest-namespace
	<When finished> <Ctrl+c> for cancel

11. Set resource for Deployment to Normal "1CPU", "1GB memory" by command:
kubectl set resources deployment/webtest --limits=cpu="1",memory=1Gi --requests=cpu="1",memory=1Gi --record --namespace=webtest-namespace
kubectl rollout status deployment/webtest --namespace=webtest-namespace
kubectl get rs --namespace=webtest-namespace
kubectl get pods --namespace=webtest-namespace
kubectl describe pods <pods name> --namespace=webtest-namespace

12. Clean up lab by command:
	kubectl delete -f webtest_deploy.yml --namespace=webtest-namespace	(In case run via github: kubectl delete -f https://raw.githubusercontent.com/praparn/kubernetesLab/master/WorkShop_1.7_Resource_Management_and_HPA/webtest_deploy.yml --namespace=webtest-namespace)
	kubectl delete -f cadvisor_deploy.yml	(In case run via github: kubectl delete -f https://raw.githubusercontent.com/praparn/kubernetesLab/master/WorkShop_1.7_Resource_Management_and_HPA/cadvisor_deploy.yml)
	kubectl delete namespace/webtest-namespace
